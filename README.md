# 以声调心 | Sonic Apothecary
### 赛博中医数字疗愈实验室 (Cyberpunk TCM Digital Healing Lab)
</div>

---

## 🧪 项目愿景 (Concept)
**以声调心 (Sonic Apothecary)** 是一项融合了“电子酸性赛博 (Acid Cyberpunk)”美学与“东方禅意中医 (Oriental Zen TCM)”哲学的沉浸式交互实验。它通过创意编程与生成艺术，将传统中医的“五音疗法”数字化，引导用户完成一段从情绪诊断到数字开方的治愈旅程。

## 🪐 核心功能 (Features)

### 1. 闻诊 · 情绪脉象 (Diagnosis)
- **星云交互系统**：用户通过在愉悦度-唤醒度坐标系中点亮光点，锁定当下的内在情绪频率。
- **数字脉象反馈**：系统实时感知脉象波动，调取五音初级样本。

### 2. 药理 · 四轨同心全息罗盘 (Pharmacology)
- **高精细度罗盘 UI**：采用四轨联动交互设计（五行、五音、五脏、情志）。
- **联动同步逻辑**：基于中医“相生相克”理论，旋转罗盘即可实现人体经络与声药频率的实时对齐。
- **全息身图**：动态反馈器官能量脉动，提供深度中医药理科普。

### 3. 炼制 · 声纹显影 (Alchemy)
- **Peter de Jong 吸引子**：利用数学算法生成极其复杂的“奇怪吸引子”视觉，模拟声波在数字炼丹炉中的共振。
- **混沌之美**：实时调整参数，将抽象的音频频率转化为丝绸般的烟雾粒子动画。

### 4. 处方 · 情绪医生 (Prescription)
- **数字化工位**：交互式拖拽与选择系统，完成最终的“声纹配伍”。
- **专属处方笺**：生成带有复古未来感的数字处方，包含服用指导与治愈指导。

## ⌨️ 交互指南 (Interactions)
- **全局切换**：使用键盘 **左/右方向键 (ArrowLeft / ArrowRight)** 在四大阶段间无缝切换。
- **滑块与按钮**：鼠标点击或拖拽可控制罗盘旋转与生成动画。

## 🛠️ 技术细节 (Tech Stack)
- **前端框架**：React 19
- **样式方案**：Tailwind CSS (Modern Cyberpunk Theme)
- **视觉引擎**：HTML5 Canvas API (Generative Art / Strange Attractors)
- **图标系统**：Lucide React
- **动效设计**：CSS3 Keyframes + Glassmorphism (毛玻璃特效)

## 🚀 本地运行 (Run Locally)

**环境准备：** Node.js

1.  **安装依赖**：
    ```bash
    npm install
    ```
2.  **启动开发环境**：
    ```bash
    npm run dev
    ```
3.  **访问应用**：在浏览器中打开 `http://localhost:5173`
